## 红岩夏季安全开发考核

### 开发内容：C2服务端，参考cobalt strike

#### 使用语言Golang

#### 实现内容：

1. 并发接受和控制多个客户端
2. 支持心跳检测
3. 可以实现单条cmd命令执行
4. 可以实现自动进行简单的信息搜集
5. 可以进行文件浏览
6. 可以上传和下载文件
7. 自定义通信协议
8. 自定义通信加密

### 自定义通信协议


客户端传入内容：

| 类型         | 意义       |
| ------------ | ---------- |
| Alive\r\n    | 心跳检测   |
| Download\r\n | 文件传输   |
| 其他         | 呈现在屏幕 |

服务端传入内容：

| 类型                           | 意义                             |
| ------------------------------ | -------------------------------- |
| Cmd\r\n指令                    | 执行cmd指令                      |
| Document\r\n                   | 进入查看文件状态                 |
| get 绝对路径                   | 获得文件（需要进入查看文件状态） |
| put 绝对路径 文件大小 文件内容 | 发送文件（需要进入查看文件状态） |
| cd  绝对路径/..                | 进入目录（需要进入查看文件状态） |
| dir 绝对路径                   | 查看文件（需要进入查看文件状态） |
